# HRA Hub; Multifactor Authenticator Troubleshooting

1. Confirm the make sure you are [using the correct login screen.](https://app.takecommand.us/sign-in)

![HRA Hub Sign In screen](https://downloads.intercomcdn.com/i/o/m61j6vi5/1213258503/3145bdd3475c96e3a90588bd740a/HRA+Hub+Sign+In.png?expires=1764707400&signature=089e21fd5ae33f4abe82f92ad6b9d76e7aff7acd61ab48721a2407183b982086&req=dSImFct7lYRfWvMW1HO4zUHCWG6oDrU2WGEEBZWfdre4OzjZg4JgX%2B7kPkv6%0AtcG7WOBzObX7H2CqJwY%3D%0A)

2. Please first refresh the page.

3. We recommend using Google Chrome.

4. Let's move to an incognito tab.

- Open Chrome and click on More (three dots) in the upper right corner of your screen.
- Select New Incognito Window.
- Alternatively, you can use the keyboard shortcut Command+Shift+N on Mac or Ctrl+Shift+N on Windows.

5. You can clear your Cache and Cookies

- Open Chrome
- Copy-paste the following text into your browser's address bar: chrome://settings/clearBrowserData
- Set the Time range to All time.
- Select Cookies and other site data and Cached images and files.
- Click Clear Data.

6. If the above still does not resolve your issue, make sure all 'Auto Filling' applications are turned off. (i.e. Google/ Apple / Samsung)

7. Try a different device. (i.e. if you are using a cell phone use a desktop).
