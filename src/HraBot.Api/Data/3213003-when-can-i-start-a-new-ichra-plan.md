# When can I start a new ICHRA plan?

Are there limitations on when I can start a new HRA during the year?

An ICHRA can be started at any time during the year. [Starting a new ICHRA plan will trigger a Qualifying Event for your employees to purchase insurance plans on the individual marketplace](https://intercom.help/take-command-health/en/articles/3213001-does-offering-an-ichra-count-as-a-qualifying-event-for-my-employees-to-purchase-insurance-outside-of-open-enrollment), which provides a lot of flexibility on when you set up your plan.
