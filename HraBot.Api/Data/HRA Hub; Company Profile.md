# HRA Hub; Company Profile

The Company Profile page provides an overview of your organization's key details, including contact information, industry and location.

To view it, log into your account and select the "Settings" option from the left hand menu.

*Note: This page contains a video tutorial (43 seconds) demonstrating the Company Profile features. Visit the original article to view the video.*

## Available Fields

### General
- First Name
- Last Name
- Email
- Phone Number
- Business Structure
- Industry
- EIN

### Address
- Company Name
- Company Address
- Suite/Apt (optional)
- City
- State
- Zip code

## Editing Your Profile

If you have access to, select the "Edit" button.

Make all the changes you need to.

Click Save!
