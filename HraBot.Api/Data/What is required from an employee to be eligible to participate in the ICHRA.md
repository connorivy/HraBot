# What is required from an employee to be eligible to participate in the ICHRA?

Is insurance required to participate in ICHRA?

An employee must fall within one of the designated classes set forth by the employer, and the employee must have an [ICHRA-compliant individual insurance plan](https://intercom.help/take-command-health/en/articles/3411994-what-types-of-premiums-are-reimbursable-through-ichra) to be eligible.
