# Can I participate in my company's ICHRA if I have COBRA coverage?

Is COBRA a qualified plan for ICHRA?

Unfortunately, no.

The guidelines for ICHRA state that employees with COBRA will need to purchase a separate individual insurance plan to participate in ICHRA. You can learn more about [what plans are compliant with ICHRA in this help article](https://intercom.help/take-command-health/en/articles/3213042-what-type-of-plans-can-my-employees-get-to-qualify-for-ichra).
